top

>Usage

The top command is a utility used to display real-time information about system processes and resource usage. It provides a dynamic, constantly updated view of system performance, including CPU usage, memory usage, and process details. It is useful for monitoring and troubleshooting system performance in real time. Press q to exit.

>Synopsis

top [option]

>Options

-o [key]

-n [number]

Limits the display to the specified number of processes.

Example 2:

-zsh

>top -n 3

-stats

Specifies which columns to display in the output.

Example 3:

-zsh

>top -stats pid,cpu,command
