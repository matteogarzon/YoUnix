uptime

>Usage

The uptime command is used to display how long the system has been running, the current time, the number of users logged in, and the system load averages.

This command provides an overview of system activity over the last 1, 5, and 15 minutes, making it helpful for monitoring system performance and resource usage. System administrators often use uptime to quickly assess the health and load of a machine.

>Synopsis

uptime (macOS)
uptime [option] (Linux)

>Options

-p

Displays the uptime in a human-readable format, like "up 5 hours, 10 minutes" (Linux-only).

Example:

uptime -p

uptime -p

up 5 hours, 15 minutes

-s

Shows the date and time the system last booted (Linux-only).

-V

Prints the version of uptime (Linux-only).
